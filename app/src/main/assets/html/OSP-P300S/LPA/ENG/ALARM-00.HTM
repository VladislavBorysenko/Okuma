ALARM & ERROR LIST (CPU ALARM)

0001BUS ERROR
       This alarm is caused when the bus error exception has occurred
       in the CPU.
       When the bus error occurs, the CPU status is displayed in
       hexadecimal numbers.
       At the same time, the red LED at the CPU rack lights up.
       The bus error includes the following five types of errors:
       3)Cycle time over error
       4)ECC error
       5)Protect error
       6)Loop error
       7)Double-bus error
       Note:
       Error display is not available in the case of double-bus error.
       [Object] None
       [Code] CPU status
       [Probable Faulty Locations]
        Printed circuit boards
        Mother board
        Contact failure of card edge connector
        Contact failure of flat cable
        Switch settings
       [Measures to Take]
       Depending on the type of error encountered, check the following
       points.
       1)ECC error -> Main board, memory board
       2)Protect error -> Main board
       3)Cycle timer over error -> Judge the faulty PCB from the
         access address.
       4)Loop error -> PCB which generates the interruption in
         question; judge the interruption
         level from the status register.
       5)System bus error -> All the main boards
       6)Double-bus error -> All the PCB's
       Then, carry out the following steps.
       1)Check the switch settings.
         Make sure that the boards are free of foreign material.
         Make sure that flat cables are inserted securely.
       2)Change the PCB inserting positions to see if there is any
         change in the error status.
       3)Replace the suspicious boards.
