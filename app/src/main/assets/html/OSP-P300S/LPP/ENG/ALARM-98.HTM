9003You can't input anymore except 0 ~ 9,(,),+,-,*,/.

       or calculate this parameter value.

       of up to five nesting levels, +, -, /, * (four arithmetic
       operations), +/- (signs), and real numbers. The entered
       formula, however, includes the other characters or
       the variables.

       Enter the formula using only parentheses of up to five
       nesting levels, +, -, /, * (four arithmetic operations),
       +/- (signs), and real numbers.

9803 Beyond the range of identifier.

The specified identifier address exceeds the processing range.

The address range may be exceeded or auxiliary information may be abnormal.

[Measures to Take]

  1. Create and install a correct auxiliary information file.

  2. Specify an identifier address within the processing range.

9804 Input data overflow.

The entered data is too large or too small.

[Measures to Take]

Enter a correct value.
