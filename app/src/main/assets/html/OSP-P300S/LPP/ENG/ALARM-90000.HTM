99019Initialize failed. 
       Initialization in file loading or initialization of the window
       system failed.
       [Code]
       Hexadecimal number notation (aabbccdd)
       aa; Outline of processing
       1: Language change information file
       2: Language combination information file
       3: FWS
       4: FWSBOOT
       bb; Details of processing
       01: File retrieval
       02: File opening
       03: File closing
       04: File reading
       05: File writing
       06: Header interpretation
       07: Command reading
       08: Command interpretation
       09: System call
       10: Initialization
       11: End
       cc; Reservation
       dd; Error code in internal processing
       [Probable Faulty Locations]
       - Control software is faulty if the first byte (aa) of the code
       (aabbccdd) is 1 or 2.
       - Control software or the board (FWMB) is faulty if the first
       byte is 3 or 4.
       [Measures to Take]
       - Check the control software.
       - Replace the board (FWMB) if the displayed code indicates that
       the board may be faulty.

99303Cannot open keyboard 
       Initialization of the keyboard failed. This error could occur
       when the control pane is faulty, the FWMB is faulty, or the
       cable between the operation panel and the FWMB is defective.
       This error may also occur if the power-on timing to the operation
       panel is delayed after turning on the main power in the case where
       the operation panel power and the main power are supplied from
       difference sources.
       [Probable Faulty Locations]
       FWMB, operation panel, or the cable between the operation
       panel and the FWMB
       [Measures to Take]
       If difference power sources are used, turn on the operation panel
       immediately after turning on the main power, and see if the above
       error still occurs.  If the same power source is used, replace the
       FWMB, operation panel, or the cable connecting the operation panel
       with the FWMB, and see if the above error still occurs.
